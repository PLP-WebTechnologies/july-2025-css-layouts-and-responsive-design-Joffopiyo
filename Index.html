Article Title

This is the main article content. It takes up a larger portion of the screen on tablets and desktops, providing ample space for text and images. On mobile devices, it occupies the full width of the container, stacking on top of the sidebar. This demonstrates the use of a flexible grid layout to adapt to different screen sizes. You can add more paragraphs and images here to showcase the responsive behavior.

The grid layout makes it easy to manage both one-dimensional and two-dimensional layouts, ensuring your page looks great on any device. The content is designed to be easily readable and accessible, with proper spacing and font sizes that adjust with the viewport. This is a core tenet of responsive web design.

Sidebar

This is the sidebar. On desktops, it sits to the right of the article. On mobile, it's placed directly below the article for a clean, single-column layout. It's a great spot for additional information, links, or advertisements.

The layout is controlled by the parent `.main-content` class, which uses `display: grid` and sets its columns based on the screen size using media queries.
